# Analiza wydajności i przestojów serwerów

Prosty skrypt treningowy służący do łączenia metryk wydajnościowych maszyn wirtualnych z ich metadanymi środowiskowymi w celu wykrywania przeciążeń systemów produkcyjnych. Program oblicza wolne zasoby CPU/RAM, grupuje dane według nazw serwerów i generuje raporty o średniej efektywności infrastruktury.
